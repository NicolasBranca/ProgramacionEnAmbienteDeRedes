Declaraciones en bloque definidas entre etiquetas style

  h1 {
    font-size: 2em;
    color: darkblue; 
    background-color: aquamarine;
    text-align: center;
  }

Todos los textos con etiqueta h1 tienen este estilo.

La etiqueta h1 ocupa el ancho del contenedor (por defecto lleva display de bloque).

La etiqueta h1 lleva color azul.

El background de un h1 lleva color aquamarine.

El estilo definido para la etiqueta h1 indica que el texto irá centrado.

  p {
    color: blue;
    font-style: italic;
    text-align: left;
    font-size: 1em;
    border-top-style: solid;
    border-bottom-style: solid;
    border-width: thin;
    border-color: blue;
  }

Todos los textos con etiquetas p tienen este estilo definido dentro de la etiqueta style.

Para los párrafos se define un estilo con estas características: alineación a la izquierda

Tamaño de letra de la misma medida que la definida para el contenedor body

Color de letra azul

Fuente de la fuente itálico

Bordes superior e inferior sólidos, finos y de color azul
