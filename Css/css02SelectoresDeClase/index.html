Selectores de clase

A partir de esta página no usaremos más el atributo style en ninguna etiqueta

Todo el estilo aplicado en el documento será definido dentro de una etiqueta style como "clase", anidada a la etiqueta head del documento.

p {
  font-size: 1em;
  margin-bottom: 60px;
}

Todos los párrafos tienen un default como estilo que incluyen un margin-bottom para separarlos entre ellos
